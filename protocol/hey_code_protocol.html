隐私协议

欢迎使用我们的应用程序 哎码（以下简称“应用”）。我们非常重视您的隐私权，并致力于保护您的个人信息。本隐私协议旨在帮助您了解我们如何收集、使用、存储和披露您的个人信息。

1. 信息收集与使用

1.1 相机权限：我们的应用可能需要访问您的设备相机，以便您可以使用应用的特定功能，如拍照或扫描二维码。相机权限将仅在您明确同意并使用这些功能时才会被请求。

1.2 相机图像：在您使用应用的相机功能时，应用可能会捕获和保存您的相机图像。这些图像可能用于应用的特定功能，但不会用于其他目的，除非您明确同意。

2. 信息保护与存储

2.1 我们将采取合理的安全措施来保护您的个人信息，以防止未经授权的访问、使用或披露。然而，我们无法绝对保证互联网传输的安全性，因此您使用我们的应用时需自行承担风险。

2.2 我们仅会在必要时保留您的个人信息，并在不再需要时进行删除或匿名化处理。

3. 共享与披露

3.1 我们不会向第三方出售、出租或交换您的个人信息。但在以下情况下，我们可能会与第三方共享您的信息：

  • 当您明确同意共享。
  • 当法律要求或允许我们共享。
  • 为了保护我们的权利、财产或安全，以及其他用户的权利、财产或安全。

4. 权利和选择

4.1 您可以随时访问和修改您的个人信息，以及撤回您的同意。

4.2 您可以关闭相机权限或删除应用，但这可能会影响应用的功能。

5. 隐私政策变更

5.1 我们可能会不时更新本隐私协议，以反映应用程序或法律的变化。任何重大变更将通过应用通知您。
